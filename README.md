# Do_An_TN-PlantCare-
Ứng dụng hỗ trợ nông nghiệp sử dụng AI để nhận diện bệnh trên cây trồng từ hình ảnh do người dùng tải lên. Hệ thống phân tích ảnh và trả về thông tin về bệnh, nguyên nhân, nguyên tố dinh dưỡng cần bổ sung và gợi ý hoạt chất thuốc. Ngoài ra, ứng dụng còn cung cấp cập nhật dịch bệnh và tin tức thị trường nông sản.
